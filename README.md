# Corn

Team project with aim to create an open source developer collaboration platform for small teams.

## Tech stack

1. Frontend

* Angular 16,
* TypeScript,
* Scss,
* Angular Material

2. Backend

* Java 17,
* Spring Boot 3.15,
* Hibernate,
* Postgres,
* JavaMailSender,
* OAuth2,
* KeyCloak,
* Mapstruct,
* Lombok
